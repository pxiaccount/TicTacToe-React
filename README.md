# TicTacToe-React

Tic Tac Toe game with React

## Installation

Clone the repository:
    ```
    git clone https://github.com/pxiaccount/TicTacToe-React.git
    ```

1. Change directory in to client:

    ```bash
    cd TicTacToe-React
    ```

2. Install packages:

    ```bash
    npm i
    ```

3. Run the development server:

    ```bash
    npm run dev
    ```
